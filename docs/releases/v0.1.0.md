# mcp-seatbelt v0.1.0

First public release.

`mcp-seatbelt` wraps stdio MCP servers with local runtime checks, JSONL audit logs, secret redaction, and a dashboard for reviewing tool calls.

## Highlights

- Wrap existing MCP servers with `mcp-seatbelt wrap -- <server command>`.
- Block high-risk tool calls for sensitive paths, destructive shell, metadata endpoints, traversal, and tool schema rug pulls.
- Redact common API tokens, bearer tokens, private keys, URL tokens, and env secret values before logs are written.
- Inspect runs with a local dashboard.
- Diagnose common MCP client configs with `mcp-seatbelt doctor`.

## Install

```bash
npx mcp-seatbelt wrap -- npx @modelcontextprotocol/server-filesystem ~/projects
```
