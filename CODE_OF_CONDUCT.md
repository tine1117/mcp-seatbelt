# Code of Conduct

This project is for practical, respectful collaboration.

## Expected Behavior

- Be direct and constructive.
- Assume good intent when asking or answering questions.
- Keep bug reports and reviews focused on reproducible behavior.
- Respect different experience levels with MCP, security tooling, and TypeScript.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Publishing private security details before maintainers have had time to respond.
- Repeated off-topic comments that make issue or review threads hard to use.

## Enforcement

Maintainers may edit, hide, or remove comments and may limit participation when behavior harms the project. If you see a problem, contact the maintainers through the repository's security or moderation channel.
